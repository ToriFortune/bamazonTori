# bamazonTori
# Node.js & MySQL

## Overview BAMAZON	
This app will function as an on line store. Customer orders are entered and the online inventory is depleted as orders are completed. Items for sale will be displayed to include the following;
-	Item name
-	Item ID
-	Item prices
-   Inventory
-   Department
